# Bloques y Sensores

Proyecto activo para analizar datos de bloques, sensores WIGGA, cortinas,
EcoWitt, Apogee y analisis de apertura.

Por ahora los datos y scripts activos siguen en las carpetas principales:

- `Datos/`
- `procesamiento/`
- `docs/`

Cuando la pagina crezca, podemos mover este proyecto completo a esta carpeta
sin perder el historial ni romper el flujo actual.
