# Proyectos

Este repositorio puede contener varios proyectos de analisis y visualizacion.

## Proyecto activo

- `bloques_y_sensores`: analisis de bloques, sensores, cortinas y variables
  ambientales.

## Proyecto futuro

- `accrel_y_reservorios`: espacio reservado para trabajar mas adelante.

La pagina principal de Streamlit podra integrar estos proyectos como secciones
o paginas independientes.
